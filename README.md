# Conference Presentations

A public speaking archive for Micah Miner's conference presentations, keynotes, workshops, and interactive scholarly sessions.

The site organizes the presentations as an evolving public argument rather than a collection of unrelated slide decks. Visitors can explore the work by audience and see how shared ideas adapt across practitioner, scholarly, and civic settings.

## View locally

Open `index.html` directly in a browser or serve the folder with any simple static web server.

## Publish

This site is designed for GitHub Pages and requires no build step or external dependencies.

## Privacy

The repository contains only public presentation links and public professional information. It contains no participant, student, staff, or research-subject data.
